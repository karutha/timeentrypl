# Deploy to Streamlit

To run this application in Streamlit:

1.  **Install Dependencies**
    ```bash
    npm install
    pip install streamlit
    ```

2.  **Build the Application**
    ```bash
    npm run build
    ```

3.  **Run Streamlit**
    ```bash
    streamlit run app.py
    ```

The application will be available at the URL shown in the terminal (usually `http://localhost:8501`).
